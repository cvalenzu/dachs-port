How to cite

Reference an underlying Publication

This service has an associated publication. If you use its data, it may be appropriate to reference source (http://adsabs.harvard.edu/cgi-bin/nph-bib_query?bibcode=&data_type=BIBTEX&db_key=AST&nocookieset=1ADS BibTeX entry for the publication) either in addition to or instead of the service reference.

Reference the Service

To cite the service as such, we suggest the following BibTeX entry:

			

			
				
The creators of this services did not provide essential
				metadata (creationDate, title, creatorName) to make a reference
				to this service; this probably means that this service is
				not intended to be referenced.  If this seems not right
				to you, please contact the data center operators.
